# Rock-Paper-Scissors-Lizard-Spock (C)
## How to Play

1. Run the program.
2. You will see a menu:
   - **1** → Play
   - **2** → Quit
3. If you choose *Play*, the rules will be shown:
   - `0` → Rock  
   - `1` → Paper  
   - `2` → Scissors  
   - `3` → Lizard  
   - `4` → Spock  
4. Enter the number for your move.
5. The computer randomly picks its move.
6. The winner of each round is shown, and the score is updated.
7. The first to reach 3 points wins the match.

---

## Winning Rules (simplified)

- Rock beats Scissors, Lizard  
- Paper beats Rock, Spock  
- Scissors beats Paper, Lizard  
- Lizard beats Spock, Paper  
- Spock beats Scissors, Rock
